> Style and Design Copyright of Pixelex Aspect and Authored By Gemma Black

## Development

### Requirements

- Docker and docker-compose
- Bash

### Installation

```bash
# Start the Docker container
./uposcar

# Composer commands
./uposcar composer install

# Npm dependencies and asssets
./uposcar npm install
./uposcar npx webpack
```
